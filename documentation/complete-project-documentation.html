! Interface Configuration
interface GigabitEthernet0/0
 nameif outside
 security-level 0
 ip address 203.0.113.2 255.255.255.252
!
interface GigabitEthernet0/1
 nameif inside
 security-level 100
 ip address 10.10.10.1 255.255.255.0
!
! NAT Configuration
nat (inside,outside) source dynamic any interface
!
! DHCP for Management VLAN
dhcpd address 10.10.10.100-10.10.10.200 inside
dhcpd enable inside

access-list outside_access_in extended permit tcp any host 203.0.113.2 eq 443
access-group outside_access_in in interface outside

🔧 PREVENTIVE MAINTENANCE SCHEDULE

Daily Checks

✅ Check NVR recording status
✅ Verify all cameras are online
✅ Check UPS status lights
✅ Review security log for any anomalies

📋 Test all emergency systems

Quarterly Checks
