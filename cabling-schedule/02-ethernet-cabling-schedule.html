🟢 GREEN - Data/Network (ONT to Router, Switch to Switch)

🟡 YELLOW - PoE/Cameras (Power + Data)

🔵 BLUE - Management/MDF Connections

🔴 RED - Critical/Cross Connect

🟣 PURPLE - VoIP Phones

🟠 ORANGE - WAN/ISP Connections

✅ CABLE TESTING REQUIREMENTS

Test Standard: TIA-568.2-D / ISO 11801 Class E

Required Tests per cable:

• Wire Map (Continuity) - 100%

• Insertion Loss (Attenuation) - Max 21.7dB @ 250MHz

• NEXT (Near End Crosstalk) - Min 35.3dB @ 250MHz

• PSNEXT - Min 32.3dB @ 250MHz

• ACR-F (Attenuation to Crosstalk Ratio) - Min 11.1dB @ 250MHz

• Return Loss - Min 19.3dB @ 250MHz

• Propagation Delay - Max 548ns

• Delay Skew - Max 45ns
